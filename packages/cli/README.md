# create-electron-app

Quickly create an electron-based app and help build the app

## Install

```sh

npm install  @zzhaon/create-electron-app --save -d

pnpm add  @zzhaon/create-electron-app --save-dev

```

## Usage

```json

    // package.json

    "script": {
        "dev": "cea",
        "build": "cea build",
    }

```
